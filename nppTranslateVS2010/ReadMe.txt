In order to build this project:

-	Get you appId by registering free with microsoft live.
-	Create a file named appId.h in this folder.
-	Open that file with notepad and copy following line in it:

# define APP_ID L"<your app id>"

-	Replace <your app id> with the one you got in step 1.
- 	Save and close the file.
-	Open .\TranslatePlugin_Rel\Translate.sln with VS2010 and build.


TIP: Run Visual Studio as admin if running on Win7
